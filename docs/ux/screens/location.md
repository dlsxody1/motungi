# 동네 설정 — `/location`

## JTBD & 목적
"어느 동네 기준으로 찾아줄지" 집 앵커를 최소 마찰로 지정한다. 거리 스코어링의 공간 축 입력.

## 레이아웃 & 위계
헤더 → "현재 위치로 찾기" 카드(geolocation) → 수동 동네 선택(검색 + 칩). 확정 시 `home` 앵커 세팅 → `/diagnosis`.
- 하드코딩 동네 5개: 망원·성수·연남·판교·합정 (`NEIGHBORHOOD_POINTS`).

## 상태
- loaded / 부분(위치 조회 중) / empty(검색 결과 없음 "곧 더 많은 동네를 추가할게요").
- **error**: geolocation 실패가 `alert()`(모바일 `location.tsx:38,53`; 웹 `location/page.tsx:42,62`) → 결함.

## 결함 (심각도)
- **🔴 C2** 확정 배너가 모든 동에 "서울 마포구 {동}" 하드코딩(모바일 `location.tsx:129-131`). 성수동(성동구)·판교동(경기 성남)에 **잘못된 정보**. → 동별 실제 구 매핑, 없으면 구 표기 생략.
- **🟠 M8** 위치 권한/실패 `alert()` → 인라인 에러 + "직접 선택" 유도로 대체.

## UX 카피
- 에러: 무엇이 일어났는지 + 어떻게(수동 선택). "위치를 가져올 수 없어요. 아래에서 동네를 직접 골라주세요."
