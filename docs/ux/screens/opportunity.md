# 활동 상세 — `/opportunity?id=`

## JTBD & 목적
"이거 나한테 왜 맞는지" 납득하고 실제 참여(외부 링크·공유)로 넘어간다. 전환 지점.

## 레이아웃 & 위계
상단바(뒤로·공유) → 카테고리 태그·제목·위치 → 참가비 카드 → "왜 맞을까요"(whyReasons, ≥1) → 메타 3칸 → "즐기는 방법" 스텝 → 집계자 고지 → 하단 액션바(보관 토글 + "보러 가기").

## 상태
- loaded. id 폴백 체인(catalog→findOpportunity→catalog[0]→ONE_PICK).
- **loading/error 부재**: 결함 M9.
- **비활성 CTA = 암묵적 unavailable 상태**: `ctaUrl` "#"/빈값 → "링크 준비 중" 비활성.

## 결함 (심각도)
- **🟠 M7** 대표 `ONE_PICK.ctaUrl:"#"` → 플래그십 상세 주 CTA가 **영구 "링크 준비 중"**. 첫인상 화면에서 죽은 주 액션(Peak-End 악화). → 실제 링크 채우거나 링크 있는 아이템으로 교체.
- **🔴 C4** 참가비 카드 side_job 의미 충돌(원픽과 동일).

## UX 카피
- "왜 {name}님께 맞을까요" 근거는 항상 ≥1 반환(폴백 보장) ✅.
- 비활성 CTA는 이유 표기: "링크 준비 중"보다 "곧 예약 링크가 열려요" 등 상태 명확화.
